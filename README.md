wombat-ui
=========

Clinical Form Builder app for Pharmaceutical Trials Data Collection. 

Demo available at:  [http://wombat-ui.meteor.com](http://wombat-ui.meteor.com)  


============================
### Installation

````sh
git clone https://github.com/awatson1978/wombat-ui.git
cd wombat-ui
sudo mrt
````
